    **3. volt/utils.ts** (3x as any — function parameters):
    - **Line 4:** `globalPTProps = {} as any, selfPTProps = {} as any, datasets: any`
    - PrimeVue PassThrough props are unstructured objects with an optional `class` property. There is no exported `PassThroughOptions` type from PrimeVue that fits these individual prop arguments. The safe replacement:
      ```typescript
      type PTProps = Record<string, unknown> & { class?: string }
      export const ptViewMerge = (
        globalPTProps: PTProps = {} as PTProps,
        selfPTProps: PTProps = {} as PTProps,
        datasets: Record<string, unknown> = {}
      ) => {
        const { class: globalClass, ...globalRest } = globalPTProps as { class?: string; [key: string]: unknown };
        const { class: selfClass, ...selfRest } = selfPTProps as { class?: string; [key: string]: unknown };
        return mergeProps({ class: twMerge(globalClass, selfClass) }, globalRest, selfRest, datasets);
      };
      ```
    - The destructuring `{ class: globalClass, ...globalRest }` requires the intermediate cast because `Record<string, unknown>` doesnt narrow `class` to `string | undefined` automatically. Use an inline type assertion to `{ class?: string; [key: string]: unknown }` which is a known-shape cast, NOT `as any`.
